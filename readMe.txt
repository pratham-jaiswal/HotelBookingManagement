REQUIREMENTS:
1. Download & Install: XAMPP in C:\xampp (default)
2. Clone this repository in C:\xampp\htdocs
3. Run XAMPP
4. On "Apache" and "MySQL" row click "start"
5. In your browser type "http://localhost/HotelBookingManagement/"
6. Open the files.

config.php - Configure Database
login.php - Allows user to Login
register.php - Allows user to create an account
accountInfo.php - If user logins in successfully, then redirect to Account Info page
logout.php - Allows user to logout
